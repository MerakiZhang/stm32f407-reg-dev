# LED 驱动说明

## 硬件连接

| 标识 | 引脚 | 有效电平 |
|------|------|----------|
| LED0 | PF9  | 低电平点亮 |
| LED1 | PF10 | 低电平点亮 |

---

## 寄存器配置说明

### 第一步：使能 GPIOF 时钟

寄存器：**RCC_AHB1ENR**（RCC AHB1 外设时钟使能寄存器）

| 位   | 名称    | 说明 |
|------|---------|------|
| 位 5 | GPIOFEN | 置 1 使能 GPIOF 时钟，置 0 关闭 |

```
RCC->AHB1ENR |= (1 << 5);
```

未使能时钟前对 GPIOF 任何寄存器的写操作均无效。

---

### 第二步：配置引脚模式（MODER）

寄存器：**GPIOF_MODER**（GPIO 端口模式寄存器）

每个引脚占 2 位，引脚 n 对应位 [2n+1 : 2n]：

| 位值 | 模式 |
|------|------|
| 00   | 输入模式 |
| 01   | 通用输出模式 |
| 10   | 复用功能模式 |
| 11   | 模拟模式 |

- PF9 对应位 [19:18]，设置为 **01**
- PF10 对应位 [21:20]，设置为 **01**

操作方式：先清零对应位，再写入 01：
```
GPIOF->MODER &= ~((3U << (9 * 2)) | (3U << (10 * 2)));   // 清零
GPIOF->MODER |=  ((1U << (9 * 2)) | (1U << (10 * 2)));   // 写入 01
```

---

### 第三步：配置输出类型（OTYPER）

寄存器：**GPIOF_OTYPER**（GPIO 端口输出类型寄存器）

每个引脚占 1 位，引脚 n 对应位 n：

| 位值 | 输出类型 |
|------|----------|
| 0    | 推挽输出（Push-Pull） |
| 1    | 开漏输出（Open-Drain） |

- PF9 对应位 9，设置为 **0**（推挽）
- PF10 对应位 10，设置为 **0**（推挽）

```
GPIOF->OTYPER &= ~((1U << 9) | (1U << 10));
```

---

### 第四步：配置输出速度（OSPEEDR）

寄存器：**GPIOF_OSPEEDR**（GPIO 端口输出速度寄存器）

每个引脚占 2 位，引脚 n 对应位 [2n+1 : 2n]：

| 位值 | 速度 |
|------|------|
| 00   | 低速（2MHz） |
| 01   | 中速（25MHz） |
| 10   | 高速（50MHz） |
| 11   | 超高速（100MHz） |

LED 驱动对速度无要求，PF9 位 [19:18]、PF10 位 [21:20] 均设置为 **00**：
```
GPIOF->OSPEEDR &= ~((3U << (9 * 2)) | (3U << (10 * 2)));
```

---

### 第五步：配置上下拉（PUPDR）

寄存器：**GPIOF_PUPDR**（GPIO 端口上下拉寄存器）

每个引脚占 2 位，引脚 n 对应位 [2n+1 : 2n]：

| 位值 | 配置 |
|------|------|
| 00   | 无上下拉 |
| 01   | 上拉 |
| 10   | 下拉 |
| 11   | 保留 |

推挽输出模式下，上下拉不影响输出电平，PF9 位 [19:18]、PF10 位 [21:20] 设置为 **00**：
```
GPIOF->PUPDR &= ~((3U << (9 * 2)) | (3U << (10 * 2)));
```

---

### 第六步：设置初始电平（BSRR）

寄存器：**GPIOF_BSRR**（GPIO 端口位设置/复位寄存器）

BSRR 是只写寄存器，分为高低两个 16 位区域：

| 位域    | 名称  | 说明 |
|---------|-------|------|
| 位 [15:0]  | BS[n] | 写 1 将引脚 n 输出置高（Set） |
| 位 [31:16] | BR[n] | 写 1 将引脚 n 输出置低（Reset） |

写 0 无效，BS 和 BR 同时置 1 时 BS 优先。

初始化完成后将 PF9、PF10 置高，默认熄灭 LED：
```
GPIOF->BSRR = (1U << 9) | (1U << 10);   // BS9、BS10 置 1
```

---

## 函数说明

### led_init

初始化 PF9、PF10 为通用推挽输出、低速、无上下拉，默认熄灭两颗 LED。

### led_on(uint8_t led)

点亮指定 LED。操作 BSRR 的 BR 位（高 16 位）将引脚拉低：
- `LED0`：写 BSRR 位 25（BR9），PF9 输出低电平
- `LED1`：写 BSRR 位 26（BR10），PF10 输出低电平

### led_off(uint8_t led)

熄灭指定 LED。操作 BSRR 的 BS 位（低 16 位）将引脚拉高：
- `LED0`：写 BSRR 位 9（BS9），PF9 输出高电平
- `LED1`：写 BSRR 位 10（BS10），PF10 输出高电平

### led_toggle(uint8_t led)

翻转指定 LED 状态。读取 ODR 对应位当前值，通过异或操作取反：
- `LED0`：`GPIOF->ODR ^= (1U << 9)`
- `LED1`：`GPIOF->ODR ^= (1U << 10)`
