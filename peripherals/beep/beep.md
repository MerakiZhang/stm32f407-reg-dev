# 蜂鸣器驱动说明

## 硬件连接

| 标识 | 引脚 | 有效电平 |
|------|------|----------|
| BEEP | PF8  | 高电平打开，低电平关闭 |

---

## 寄存器配置说明

### 第一步：使能 GPIOF 时钟

寄存器：**RCC_AHB1ENR**（RCC AHB1 外设时钟使能寄存器）

| 位   | 名称    | 说明 |
|------|---------|------|
| 位 5 | GPIOFEN | 置 1 使能 GPIOF 时钟，置 0 关闭 |

```
RCC->AHB1ENR |= (1 << 5);
```

---

### 第二步：配置引脚模式（MODER）

寄存器：**GPIOF_MODER**（GPIO 端口模式寄存器）

每个引脚占 2 位，引脚 n 对应位 [2n+1 : 2n]：

| 位值 | 模式 |
|------|------|
| 00   | 输入模式 |
| 01   | 通用输出模式 |
| 10   | 复用功能模式 |
| 11   | 模拟模式 |

PF8 对应位 [17:16]，设置为 **01**（通用输出）：
```
GPIOF->MODER &= ~(3U << (8 * 2));   // 清零位 [17:16]
GPIOF->MODER |=  (1U << (8 * 2));   // 写入 01
```

---

### 第三步：配置输出类型（OTYPER）

寄存器：**GPIOF_OTYPER**（GPIO 端口输出类型寄存器）

每个引脚占 1 位，引脚 n 对应位 n：

| 位值 | 输出类型 |
|------|----------|
| 0    | 推挽输出（Push-Pull） |
| 1    | 开漏输出（Open-Drain） |

PF8 对应位 8，设置为 **0**（推挽）：
```
GPIOF->OTYPER &= ~(1U << 8);
```

---

### 第四步：配置输出速度（OSPEEDR）

寄存器：**GPIOF_OSPEEDR**（GPIO 端口输出速度寄存器）

每个引脚占 2 位，引脚 n 对应位 [2n+1 : 2n]：

| 位值 | 速度 |
|------|------|
| 00   | 低速（2MHz） |
| 01   | 中速（25MHz） |
| 10   | 高速（50MHz） |
| 11   | 超高速（100MHz） |

蜂鸣器对速度无要求，PF8 位 [17:16] 设置为 **00**：
```
GPIOF->OSPEEDR &= ~(3U << (8 * 2));
```

---

### 第五步：配置上下拉（PUPDR）

寄存器：**GPIOF_PUPDR**（GPIO 端口上下拉寄存器）

每个引脚占 2 位，引脚 n 对应位 [2n+1 : 2n]：

| 位值 | 配置 |
|------|------|
| 00   | 无上下拉 |
| 01   | 上拉 |
| 10   | 下拉 |
| 11   | 保留 |

PF8 位 [17:16] 设置为 **00**：
```
GPIOF->PUPDR &= ~(3U << (8 * 2));
```

---

### 第六步：设置初始电平（BSRR）

寄存器：**GPIOF_BSRR**（GPIO 端口位设置/复位寄存器）

BSRR 是只写寄存器，分为高低两个 16 位区域：

| 位域       | 名称  | 说明 |
|------------|-------|------|
| 位 [15:0]  | BS[n] | 写 1 将引脚 n 输出置高（Set） |
| 位 [31:16] | BR[n] | 写 1 将引脚 n 输出置低（Reset） |

初始化完成后将 PF8 置低，默认关闭蜂鸣器：
```
GPIOF->BSRR = (1U << (8 + 16));   // BR8 置 1，PF8 输出低电平
```

---

## 函数说明

### beep_init

初始化 PF8 为通用推挽输出、低速、无上下拉，默认输出低电平，蜂鸣器关闭。

### beep_on

打开蜂鸣器。操作 BSRR 的 BS 位（低 16 位）将 PF8 置高：
```
GPIOF->BSRR = (1U << 8);   // BS8 置 1，PF8 输出高电平
```

### beep_off

关闭蜂鸣器。操作 BSRR 的 BR 位（高 16 位）将 PF8 置低：
```
GPIOF->BSRR = (1U << (8 + 16));   // BR8 置 1，PF8 输出低电平
```

### beep_toggle

翻转蜂鸣器状态。读取 ODR 位 8 当前值，通过异或操作取反：
```
GPIOF->ODR ^= (1U << 8);
```
