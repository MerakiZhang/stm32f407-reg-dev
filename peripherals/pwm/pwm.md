# PWM 驱动说明

## 功能概述

使用 TIM14 通用定时器的输出比较（PWM）功能，通过硬件自动翻转 PF9 引脚电平，控制 LED0 亮度。主循环只需调用 `pwm_set_duty()` 更新占空比，无需 ISR 介入。

---

## 引脚复用

| 引脚 | 复用功能 | AF 编号 | 定时器通道 |
|------|----------|---------|------------|
| PF9  | TIM14_CH1 | AF9 | TIM14 通道 1 |

GPIO 需配置为**复用功能模式**（MODER = 10），并在 AFRH 中写入 AF9。

---

## 时钟来源

TIM14 挂载在 APB1 总线（同 TIM3）。APB1 分频系数 = 4 ≠ 1，因此 TIM 时钟 = 2 × APB1。

| 时钟 | 频率 |
|------|------|
| SYSCLK | 168MHz |
| APB1   | 42MHz（÷4） |
| TIM14  | **84MHz**（2 × APB1） |

---

## PWM 参数

```
f_PWM = TIM14_CLK / (PSC + 1) / (ARR + 1)
      = 84MHz / 84 / 1000 = 1kHz
```

| 参数 | 值 | 说明 |
|------|----|------|
| PSC  | 83 | 计数频率 = 84MHz / 84 = 1MHz |
| ARR  | 999 | PWM 周期 = 1000 个 tick = 1ms（1kHz） |
| CCR1 | 0 ~ 1000 | 占空比控制值 |

---

## 占空比与 CCR1 的关系

LED0 为**低电平点亮**（Active-Low）。

**PWM 模式 1** 行为：
- 计数值 **< CCR1** → 输出**高电平**（LED 灭）
- 计数值 **≥ CCR1** → 输出**低电平**（LED 亮）

低电平（LED 亮）占比 = (ARR + 1 − CCR1) / (ARR + 1)

| duty（%） | CCR1 | 效果 |
|-----------|------|------|
| 0   | 1000 | 始终高电平，LED 灭 |
| 50  | 500  | 50% 低电平，LED 半亮 |
| 100 | 0    | 始终低电平，LED 全亮 |

换算公式：`CCR1 = (100 − duty) × 10`

---

## 涉及寄存器

### GPIOF_MODER — 引脚模式

PF9 需设为复用功能模式（bits[19:18] = 10）。

```c
GPIOF->MODER &= ~(3U << 18);
GPIOF->MODER |=  (2U << 18);   /* AF 模式 */
```

### GPIOF_AFRH — 复用功能选择（高位）

PF9 对应 AFRH bits[7:4]，写入 9 选择 AF9（TIM14_CH1）。

```c
GPIOF->AFR[1] &= ~(0xFU << 4);
GPIOF->AFR[1] |=  (9U   << 4);   /* AF9 */
```

### RCC_APB1ENR — TIM14 时钟使能

| 位   | 名称    | 说明 |
|------|---------|------|
| 位 8 | TIM14EN | 置 1 使能 TIM14 时钟 |

```c
RCC->APB1ENR |= (1U << 8);
```

### TIMx_CCMR1 — 输出比较模式寄存器

| 位域   | 名称   | 值 | 说明 |
|--------|--------|----|------|
| [6:4]  | OC1M   | 110 | PWM 模式 1 |
| [3]    | OC1PE  | 1  | 输出比较预装载使能 |

```c
TIM14->CCMR1 = (6U << 4) | (1U << 3);
```

### TIMx_CCER — 输出使能

| 位   | 名称 | 说明 |
|------|------|------|
| 位 0 | CC1E | 置 1 使能 CH1 输出 |
| 位 1 | CC1P | 0 = 高电平有效（不取反） |

```c
TIM14->CCER |= (1U << 0);
```

### TIMx_CCR1 — 输出比较值

写入该寄存器即可实时更新占空比（OC1PE=1 时下一更新事件生效）。

```c
TIM14->CCR1 = 500;   /* 50% 亮度 */
```

### TIMx_CR1 — 控制寄存器 1

| 位   | 名称 | 说明 |
|------|------|------|
| 位 7 | ARPE | ARR 预装载使能 |
| 位 0 | CEN  | 计数器使能 |

```c
TIM14->CR1 |= (1U << 7) | (1U << 0);
```

---

## 配置步骤

1. **GPIO**：PF9 → AF 模式，写入 AFRH AF9
2. **使能 TIM14 时钟**：RCC_APB1ENR bit8
3. **设置 PSC 和 ARR**：确定 PWM 频率
4. **配置 CCMR1**：PWM 模式 1 + OC1PE
5. **使能输出**：CCER CC1E
6. **产生更新事件**：EGR UG，使 PSC/ARR 装入影子寄存器
7. **设置初始 CCR1**：1000（LED 灭）
8. **启动计数器**：CR1 ARPE + CEN

> TIM14 硬件 PWM 无需配置 NVIC，也无需编写 ISR。

---

## 函数说明

### pwm_init

```c
void pwm_init(void);
```

初始化 PF9 复用功能和 TIM14 PWM 输出。初始占空比为 0（LED0 熄灭）。

### pwm_set_duty

```c
void pwm_set_duty(uint8_t duty);
```

设置 LED0 亮度。

| 参数 | 说明 |
|------|------|
| duty | 0（灭）~ 100（全亮），超出范围自动限幅 |

调用示例：

```c
pwm_set_duty(0);     /* 熄灭 */
pwm_set_duty(50);    /* 半亮 */
pwm_set_duty(100);   /* 全亮 */
```

---

## 呼吸灯示例

```c
/* 渐亮：0 → 100，每步 10ms */
for (uint8_t d = 0; d <= 100; d++)
{
    pwm_set_duty(d);
    delay_ms(10);
}
/* 渐暗：99 → 0，每步 10ms */
for (int16_t d = 99; d >= 0; d--)
{
    pwm_set_duty((uint8_t)d);
    delay_ms(10);
}
```

总周期 = 200 步 × 10ms = **2s**。
