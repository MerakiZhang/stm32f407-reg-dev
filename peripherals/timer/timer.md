# 通用定时器驱动说明

## 功能概述

使用 TIM3 通用定时器产生周期性更新中断。计数器从 0 递增到 ARR，溢出后重置并触发更新中断（Update Interrupt），ISR 内置位标志变量，主循环检测标志后执行对应动作。

---

## 定时器时钟来源

TIM3 挂载在 APB1 总线。STM32F407 规定：当 APB1 分频系数不为 1 时，TIM 时钟 = 2 × APB1 时钟。

| 时钟 | 频率 | 说明 |
|------|------|------|
| SYSCLK | 168MHz | PLL 输出 |
| APB1 | 42MHz | SYSCLK / 4 |
| TIM3 时钟 | **84MHz** | 2 × APB1（分频系数 = 4 ≠ 1） |

---

## 中断周期计算

```
T = (PSC + 1) × (ARR + 1) / f_TIM3
```

| 参数 | 说明 |
|------|------|
| PSC  | 预分频值（16 位），计数频率 = 84MHz / (PSC + 1) |
| ARR  | 自动重载值（16 位），计数器从 0 计到 ARR 后溢出 |

**示例（500ms）：**

```
PSC = 8399  →  f_cnt = 84MHz / 8400 = 10kHz（每 tick 0.1ms）
ARR = 4999  →  T = 5000 × 0.1ms = 500ms（2Hz 中断频率）
```

---

## 涉及寄存器

### RCC_APB1ENR — APB1 外设时钟使能

| 位   | 名称   | 说明 |
|------|--------|------|
| 位 1 | TIM3EN | 置 1 使能 TIM3 时钟 |

```c
RCC->APB1ENR |= (1U << 1);
```

### TIMx_PSC — 预分频寄存器

16 位寄存器。计数时钟 = TIM3 时钟 / (PSC + 1)。PSC 值通过更新事件装入影子寄存器后生效。

```c
TIM3->PSC = 8399;   /* f_cnt = 84MHz / 8400 = 10kHz */
```

### TIMx_ARR — 自动重载寄存器

16 位寄存器（TIM3 实际为 32 位，但此处用 16 位参数即可）。计数器递增到 ARR 后产生溢出，触发更新事件并重置为 0。

```c
TIM3->ARR = 4999;   /* 5000 个 tick × 0.1ms = 500ms */
```

### TIMx_EGR — 事件产生寄存器

| 位   | 名称 | 说明 |
|------|------|------|
| 位 0 | UG   | 软件产生更新事件，PSC/ARR 立即装入影子寄存器，同时置位 SR.UIF |

PSC 和 ARR 写入后，需通过 EGR 产生更新事件才能立即生效；否则要等到计数器第一次自然溢出才装载。

```c
TIM3->EGR = (1U << 0);   /* 强制更新，使 PSC/ARR 立即生效 */
TIM3->SR &= ~(1U << 0);  /* 清除 EGR 写 UG 产生的 UIF，避免立即触发中断 */
```

### TIMx_SR — 状态寄存器

| 位   | 名称 | 说明 |
|------|------|------|
| 位 0 | UIF  | 更新中断标志，溢出时硬件置 1；软件**写 0 清除**（写 1 无效） |

```c
if (TIM3->SR & (1U << 0))   /* 检查 UIF */
{
    TIM3->SR &= ~(1U << 0); /* 清除 UIF（rc_w0 位：写 0 有效） */
}
```

### TIMx_DIER — DMA/中断使能寄存器

| 位   | 名称 | 说明 |
|------|------|------|
| 位 0 | UIE  | 更新中断使能，置 1 后计数器溢出时产生中断请求 |

```c
TIM3->DIER |= (1U << 0);   /* 使能更新中断 */
```

### TIMx_CR1 — 控制寄存器 1

| 位   | 名称 | 说明 |
|------|------|------|
| 位 0 | CEN  | 计数器使能，置 1 启动计数 |

```c
TIM3->CR1 |= (1U << 0);   /* 启动计数器 */
```

---

## 配置步骤

1. **使能 TIM3 时钟**：RCC_APB1ENR bit1
2. **设置 PSC 和 ARR**：确定计数频率和中断周期
3. **产生更新事件**：EGR 写 UG，使 PSC/ARR 立即装入影子寄存器
4. **清除 UIF**：防止初始化后立即触发中断
5. **使能更新中断**：DIER 置位 UIE
6. **配置 NVIC**：设置优先级，使能 TIM3_IRQn
7. **启动计数器**：CR1 置位 CEN

---

## NVIC 优先级说明

| 中断 | 优先级 | 说明 |
|------|--------|------|
| SysTick | 0 | 最高，保证毫秒计数不被阻断 |
| TIM3 | 1 | 低于 SysTick，可正常嵌套 |
| EXTI | 2 | 最低，不影响定时器精度 |

```c
NVIC_SetPriority(TIM3_IRQn, 1);
NVIC_EnableIRQ(TIM3_IRQn);
```

---

## 标志位使用方法

ISR 置位标志，主循环检测后先清零再执行动作：

```c
if (timer3_flag)
{
    timer3_flag = 0;   /* 先清零，防止主循环执行期间再次溢出被漏掉 */
    /* 执行定时任务 */
    led_toggle(LED0);
}
```

---

## 函数说明

### timer3_init

```c
void timer3_init(uint16_t psc, uint16_t arr);
```

初始化 TIM3 并启动计数。中断周期 = (psc + 1) × (arr + 1) / 84MHz。

| 参数 | 说明 |
|------|------|
| psc  | 预分频值（0~65535） |
| arr  | 自动重载值（0~65535） |

调用示例：

```c
timer3_init(8399, 4999);   /* 500ms 中断周期 */
timer3_init(8399, 9999);   /* 1000ms 中断周期 */
timer3_init(839, 999);     /* 10ms 中断周期 */
```

### TIM3_IRQHandler

TIM3 更新中断服务函数。检查 UIF 标志，清除后置位 `timer3_flag`。

# TIM5 输入捕获驱动说明

## 功能概述

使用 TIM5 通用定时器的输入捕获功能测量 PA0（KEY_UP）高电平脉宽。上升沿触发时记录计数值，下降沿触发时计算差值，结果以微秒（µs）为单位存入全局变量。

---

## 硬件映射

| 引脚 | 定时器通道 | 复用功能 |
|------|-----------|---------|
| PA0  | TIM5_CH1  | AF2     |

PA0 内部下拉由 `key_init()` 配置，`tim5ic_init()` 仅修改 MODER 和 AFR，PUPDR 不变。因此 **`tim5ic_init()` 须在 `key_init()` 之后调用**。

---

## 定时器时钟及精度

TIM5 挂载在 APB1 总线，与 TIM3 相同：

| 时钟 | 频率 | 说明 |
|------|------|------|
| SYSCLK | 168MHz | PLL 输出 |
| APB1   | 42MHz  | SYSCLK / 4 |
| TIM5 时钟 | **84MHz** | 2 × APB1 |

配置参数：

| 参数 | 值 | 说明 |
|------|----|------|
| PSC  | 83 | 84MHz / 84 = **1MHz**，1µs / tick |
| ARR  | 0xFFFFFFFF | 32 位计数器满量程，最长约 4295 秒 |

---

## 测量原理

```
PA0 电平：  _____|‾‾‾‾‾‾‾‾‾‾‾‾‾‾‾|______
                 ↑上升沿             ↓下降沿
TIM5 CCR1：      rise_val           fall_val
脉宽 = fall_val − rise_val（µs）
```

32 位无符号减法自动处理计数器溢出：即使 TIM5->CNT 在测量期间从 0xFFFFFFFF 回绕到 0，差值仍然正确。

---

## 捕获极性切换逻辑

```
初始：CC1P = 0（上升沿）
          │
     上升沿 ISR
          ├── 记录 CCR1 为 rise_val
          └── 设置 CC1P = 1（切换为下降沿）
               │
          下降沿 ISR
               ├── pulse_us = CCR1 − rise_val
               ├── pulse_ready = 1
               └── 设置 CC1P = 0（切回上升沿）
```

---

## 涉及寄存器

### GPIOA_MODER / AFR[0] — PA0 引脚配置

```c
GPIOA->MODER  &= ~(3U << 0);    /* 清除 PA0 模式位 */
GPIOA->MODER  |=  (2U << 0);    /* 10b = 复用功能模式 */
GPIOA->AFR[0] &= ~(0xFU << 0);  /* 清除 PA0 AF 选择位 */
GPIOA->AFR[0] |=  (2U << 0);    /* AF2 = TIM5_CH1 */
```

### TIMx_CCMR1 — 捕获/比较模式寄存器 1（低 8 位，CH1）

| 位域 | 名称 | 值 | 说明 |
|------|------|-----|------|
| [1:0] | CC1S | 01 | IC1 映射到 TI1（输入捕获） |
| [3:2] | IC1PSC | 00 | 不分频，每次边沿均捕获 |
| [7:4] | IC1F | 0000 | 无数字滤波 |

```c
TIM5->CCMR1 &= ~(0xFFU);
TIM5->CCMR1 |=  (1U << 0);   /* CC1S = 01 */
```

### TIMx_CCER — 捕获/比较使能寄存器

| 位   | 名称  | 说明 |
|------|-------|------|
| 位 0 | CC1E  | 1 = 使能 CH1 捕获 |
| 位 1 | CC1P  | 0 = 上升沿；1 = 下降沿 |
| 位 3 | CC1NP | 0（与 CC1P 配合确定极性） |

上升沿切下降沿（ISR 内动态切换）：

```c
TIM5->CCER |=  (1U << 1);   /* CC1P = 1 → 下降沿 */
TIM5->CCER &= ~(1U << 1);   /* CC1P = 0 → 上升沿 */
```

### TIMx_CCR1 — 捕获/比较寄存器 1

输入捕获模式下，每次触发时硬件自动将 CNT 当前值锁存至 CCR1，同时置位 SR.CC1IF。

```c
uint32_t val = TIM5->CCR1;   /* 读取捕获值（同时清除 CC1IF 的过捕获标志） */
```

### TIMx_DIER — DMA/中断使能寄存器

| 位   | 名称  | 说明 |
|------|-------|------|
| 位 1 | CC1IE | 1 = 使能 CH1 捕获中断 |

```c
TIM5->DIER |= (1U << 1);
```

### TIMx_SR — 状态寄存器

| 位   | 名称  | 说明 |
|------|-------|------|
| 位 1 | CC1IF | CH1 捕获中断标志，写 0 清除 |

```c
TIM5->SR &= ~(1U << 1);   /* 清除 CC1IF */
```

---

## 配置步骤

1. **使能 GPIOA 时钟**（`key_init()` 已完成，重复无害）
2. **PA0 → AF2（TIM5_CH1）**：MODER=10，AFR[0]=2
3. **使能 TIM5 时钟**：RCC_APB1ENR bit3
4. **设置 PSC = 83，ARR = 0xFFFFFFFF**
5. **CH1 输入捕获模式**：CCMR1 CC1S=01
6. **初始极性：上升沿**：CCER CC1P=0，CC1NP=0
7. **使能捕获和中断**：CCER CC1E=1，DIER CC1IE=1
8. **产生更新事件，清除所有标志**：EGR UG=1，SR=0
9. **配置 NVIC**：TIM5_IRQn 优先级 2
10. **启动计数器**：CR1 CEN=1

---

## NVIC 优先级汇总

| 中断 | 优先级 | 说明 |
|------|--------|------|
| SysTick | 0 | 最高，保证毫秒计数 |
| TIM3    | 1 | 周期中断 |
| TIM5    | 2 | 输入捕获，与 EXTI 同级 |
| EXTI    | 2 | 外部中断按键 |

---

## 函数说明

### tim5ic_init

```c
void tim5ic_init(void);
```

初始化 TIM5 输入捕获。须在 `key_init()` 之后调用（依赖 PA0 的下拉配置）。

### TIM5_IRQHandler

TIM5 CH1 捕获中断服务函数。通过 `tim5_edge` 状态变量区分上升/下降沿，计算脉宽后置位 `tim5_pulse_ready`。

---

## 结果读取

```c
tim5ic_init();

while (1) {
    if (tim5_pulse_ready) {
        tim5_pulse_ready = 0;                  /* 先清零，准备下次捕获 */
        uint32_t us = tim5_pulse_us;           /* 读取脉宽（µs） */
        printf("pulse width: %lu us\r\n", us);
    }
}
```

# TIM2 电容按键检测驱动说明

## 功能概述

使用 TIM2 通用定时器的输入捕获功能，通过**充放电时间测量法**检测 PA5 上的电容按键是否按下。PA5 外接上拉电阻至 3.3V，构成 RC 充电回路；手指触摸时并联体表电容，充电时间延长，通过与基线对比判定触摸状态。

---

## 硬件映射

| 引脚 | 定时器通道 | 复用功能 | 外部上拉 |
|------|-----------|---------|---------|
| PA5  | TIM2_CH1  | AF1     | 3.3V（外接） |

PA5 配置为浮空输入（PUPDR=00），内部不配置上下拉，依赖外部上拉电阻。

---

## 检测原理

```
每次检测循环：

  ① 放电阶段
     PA5 → 推挽输出 → 拉低 50µs
     （强制电容放电至 0V）

  ② 充电 + 捕获阶段
     PA5 → AF1（TIM2_CH1）
     外部上拉对电容充电 ↑
     TIM2 捕获 PA5 上升沿 → CCR1 = 充电时间（µs）

  ③ 判断
     充电时间 > 基线 + 阈值  →  触摸按下
     充电时间 ≤ 基线 + 阈值  →  未触摸
```

**触摸前后对比：**

```
未触摸：C 小 → RC 小 → 充电快 → 充电时间短（≈ 基线）
触  摸：C 增大（手指并联） → RC 增大 → 充电慢 → 充电时间 > 基线 + 阈值
```

---

## 定时器时钟及精度

TIM2 挂载在 APB1 总线，与 TIM3/TIM5 相同：

| 时钟 | 频率 | 说明 |
|------|------|------|
| SYSCLK | 168MHz | PLL 输出 |
| APB1   | 42MHz  | SYSCLK / 4 |
| TIM2 时钟 | **84MHz** | 2 × APB1 |

配置参数：

| 参数 | 值 | 说明 |
|------|----|------|
| PSC  | 83 | 84MHz / 84 = **1MHz**，1µs / tick |
| ARR  | 0xFFFFFFFF | 32 位计数器满量程 |

---

## 可调参数

| 宏定义 | 默认值 | 说明 |
|--------|--------|------|
| `TIM2_DISCHARGE_US` | 50 | 放电时长（µs），需确保完全放电 |
| `TIM2_TIMEOUT_US` | 500 | 等待上升沿超时（µs），防止上拉断开时死等 |
| `TIM2_CAP_THRESHOLD` | 20 | 触摸判定阈值（µs），超过基线多少判定为触摸 |
| `TIM2_CAL_SAMPLES` | 16 | 基线校准采样次数 |

**阈值调整建议：** 上拉电阻阻值越大、按键焊盘电容越大，充电时间越长，基线值越高，阈值可适当增大（建议为基线值的 20%~50%）。

---

## 涉及寄存器

### GPIOA_MODER / AFR[0] / PUPDR — PA5 引脚配置

```c
GPIOA->MODER  &= ~(3U << (5 * 2));    /* 清除 PA5 模式位 */
GPIOA->MODER  |=  (2U << (5 * 2));    /* 10b = 复用功能模式 */
GPIOA->AFR[0] &= ~(0xFU << (5 * 4)); /* 清除 PA5 AF 选择位 */
GPIOA->AFR[0] |=  (1U   << (5 * 4)); /* AF1 = TIM2_CH1 */
GPIOA->PUPDR  &= ~(3U << (5 * 2));   /* 00 = 浮空，依赖外部上拉 */
```

放电阶段临时切换为输出模式：

```c
GPIOA->MODER &= ~(3U << (5 * 2));
GPIOA->MODER |=  (1U << (5 * 2));    /* 01b = 输出模式 */
GPIOA->BSRR   =  (1U << (5 + 16));   /* PA5 = 0，放电 */
```

### TIMx_CCMR1 — 捕获/比较模式寄存器 1（低 8 位，CH1）

| 位域 | 名称 | 值 | 说明 |
|------|------|-----|------|
| [1:0] | CC1S | 01 | IC1 映射到 TI1（输入捕获） |
| [3:2] | IC1PSC | 00 | 不分频 |
| [7:4] | IC1F | 0000 | 无数字滤波 |

```c
TIM2->CCMR1 &= ~(0xFFU);
TIM2->CCMR1 |=  (1U << 0);   /* CC1S = 01 */
```

### TIMx_CCER — 捕获/比较使能寄存器

| 位   | 名称  | 值 | 说明 |
|------|-------|-----|------|
| 位 0 | CC1E  | 1 | 使能 CH1 捕获 |
| 位 1 | CC1P  | 0 | 上升沿捕获（充电完成时 PA5 变高） |
| 位 3 | CC1NP | 0 | 与 CC1P 配合 |

```c
TIM2->CCER &= ~((1U << 1) | (1U << 3));  /* 上升沿 */
TIM2->CCER |=  (1U << 0);               /* CC1E = 1 */
```

### TIMx_CCR1 — 捕获寄存器

充电完成（PA5 上升沿）时，TIM2 硬件自动将 CNT 当前值锁存至 CCR1，同时置位 SR.CC1IF。CNT 已在放电结束后清零，因此 CCR1 直接等于充电时间（µs）。

```c
if (TIM2->SR & (1U << 1))   /* CC1IF */
{
    uint32_t charge_time = TIM2->CCR1;
}
```

---

## 配置步骤

1. **使能 GPIOA 和 TIM2 时钟**：AHB1ENR bit0，APB1ENR bit0
2. **PA5 → AF1（TIM2_CH1）**：MODER=10，AFR[0]=1，PUPDR=00（浮空）
3. **设置 PSC=83，ARR=0xFFFFFFFF**
4. **CH1 输入捕获，上升沿**：CCMR1 CC1S=01，CCER CC1P=0，CC1E=1
5. **不使用捕获中断**：DIER=0（轮询方式）
6. **产生更新事件，清除标志**：EGR UG=1，SR=0
7. **启动计数器**：CR1 CEN=1
8. **基线校准**：采样 16 次取均值（须在无触摸状态下执行）

---

## 函数说明

### tim2cap_init

```c
void tim2cap_init(void);
```

初始化 TIM2 输入捕获，并自动完成基线校准。**调用时须保持电容按键无触摸**，校准期间约耗时 5ms。

### tim2cap_scan

```c
uint8_t tim2cap_scan(void);
```

执行一次充放电检测，更新 `tim2_cap_state` 并返回当前状态。

| 返回值 | 含义 |
|--------|------|
| 1 | 触摸按下 |
| 0 | 未触摸 |

单次调用耗时约 50µs（放电）+ 充电时间（通常 < 500µs）。建议以 **5~10ms** 为周期轮询调用。

---

## 使用示例

```c
tim2cap_init();   /* 初始化，无触摸状态下校准基线 */

uint8_t  prev = 0;
uint32_t press_tick = 0;

while (1) {
    uint8_t cap = tim2cap_scan();

    if (cap && !prev) {
        /* 按下：点亮 LED，记录时刻 */
        led_on(LED1);
        press_tick = delay_get_tick();
    } else if (!cap && prev) {
        /* 松开：熄灭 LED，打印持续时间 */
        led_off(LED1);
        printf("CAP_KEY held: %lu ms\r\n", delay_get_tick() - press_tick);
    }

    prev = cap;
}
```

# TIM14 PWM 输出驱动说明

## 功能概述

使用 TIM14 通用定时器的输出比较（PWM）功能，通过硬件自动翻转 PF9 引脚电平，控制 LED0 亮度。主循环只需调用 `pwm_set_duty()` 更新占空比，无需 ISR 介入。

---

## 引脚复用

| 引脚 | 复用功能 | AF 编号 | 定时器通道 |
|------|----------|---------|------------|
| PF9  | TIM14_CH1 | AF9 | TIM14 通道 1 |

GPIO 需配置为**复用功能模式**（MODER = 10），并在 AFRH 中写入 AF9。

---

## 时钟来源

TIM14 挂载在 APB1 总线（同 TIM3）。APB1 分频系数 = 4 ≠ 1，因此 TIM 时钟 = 2 × APB1。

| 时钟 | 频率 |
|------|------|
| SYSCLK | 168MHz |
| APB1   | 42MHz（÷4） |
| TIM14  | **84MHz**（2 × APB1） |

---

## PWM 参数

```
f_PWM = TIM14_CLK / (PSC + 1) / (ARR + 1)
      = 84MHz / 84 / 1000 = 1kHz
```

| 参数 | 值 | 说明 |
|------|----|------|
| PSC  | 83 | 计数频率 = 84MHz / 84 = 1MHz |
| ARR  | 999 | PWM 周期 = 1000 个 tick = 1ms（1kHz） |
| CCR1 | 0 ~ 1000 | 占空比控制值 |

---

## 占空比与 CCR1 的关系

LED0 为**低电平点亮**（Active-Low）。

**PWM 模式 1** 行为：
- 计数值 **< CCR1** → 输出**高电平**（LED 灭）
- 计数值 **≥ CCR1** → 输出**低电平**（LED 亮）

低电平（LED 亮）占比 = (ARR + 1 − CCR1) / (ARR + 1)

| duty（%） | CCR1 | 效果 |
|-----------|------|------|
| 0   | 1000 | 始终高电平，LED 灭 |
| 50  | 500  | 50% 低电平，LED 半亮 |
| 100 | 0    | 始终低电平，LED 全亮 |

换算公式：`CCR1 = (100 − duty) × 10`

## 涉及寄存器

### GPIOF_MODER — 引脚模式

PF9 需设为复用功能模式（bits[19:18] = 10）。

```c
GPIOF->MODER &= ~(3U << 18);
GPIOF->MODER |=  (2U << 18);   /* AF 模式 */
```

### GPIOF_AFRH — 复用功能选择（高位）

PF9 对应 AFRH bits[7:4]，写入 9 选择 AF9（TIM14_CH1）。

```c
GPIOF->AFR[1] &= ~(0xFU << 4);
GPIOF->AFR[1] |=  (9U   << 4);   /* AF9 */
```

### RCC_APB1ENR — TIM14 时钟使能

| 位   | 名称    | 说明 |
|------|---------|------|
| 位 8 | TIM14EN | 置 1 使能 TIM14 时钟 |

```c
RCC->APB1ENR |= (1U << 8);
```

### TIMx_CCMR1 — 输出比较模式寄存器

| 位域   | 名称   | 值 | 说明 |
|--------|--------|----|------|
| [6:4]  | OC1M   | 110 | PWM 模式 1 |
| [3]    | OC1PE  | 1  | 输出比较预装载使能 |

```c
TIM14->CCMR1 = (6U << 4) | (1U << 3);
```

### TIMx_CCER — 输出使能

| 位   | 名称 | 说明 |
|------|------|------|
| 位 0 | CC1E | 置 1 使能 CH1 输出 |
| 位 1 | CC1P | 0 = 高电平有效（不取反） |

```c
TIM14->CCER |= (1U << 0);
```

### TIMx_CCR1 — 输出比较值

写入该寄存器即可实时更新占空比（OC1PE=1 时下一更新事件生效）。

```c
TIM14->CCR1 = 500;   /* 50% 亮度 */
```

### TIMx_CR1 — 控制寄存器 1

| 位   | 名称 | 说明 |
|------|------|------|
| 位 7 | ARPE | ARR 预装载使能 |
| 位 0 | CEN  | 计数器使能 |

```c
TIM14->CR1 |= (1U << 7) | (1U << 0);
```

---

## 配置步骤

1. **GPIO**：PF9 → AF 模式，写入 AFRH AF9
2. **使能 TIM14 时钟**：RCC_APB1ENR bit8
3. **设置 PSC 和 ARR**：确定 PWM 频率
4. **配置 CCMR1**：PWM 模式 1 + OC1PE
5. **使能输出**：CCER CC1E
6. **产生更新事件**：EGR UG，使 PSC/ARR 装入影子寄存器
7. **设置初始 CCR1**：1000（LED 灭）
8. **启动计数器**：CR1 ARPE + CEN

> TIM14 硬件 PWM 无需配置 NVIC，也无需编写 ISR。

## 函数说明

### pwm_init

```c
void pwm_init(void);
```

初始化 PF9 复用功能和 TIM14 PWM 输出。初始占空比为 0（LED0 熄灭）。

### pwm_set_duty

```c
void pwm_set_duty(uint8_t duty);
```

设置 LED0 亮度。

| 参数 | 说明 |
|------|------|
| duty | 0（灭）~ 100（全亮），超出范围自动限幅 |

调用示例：

```c
pwm_set_duty(0);     /* 熄灭 */
pwm_set_duty(50);    /* 半亮 */
pwm_set_duty(100);   /* 全亮 */
```

---

## 呼吸灯示例

```c
/* 渐亮：0 → 100，每步 10ms */
for (uint8_t d = 0; d <= 100; d++)
{
    pwm_set_duty(d);
    delay_ms(10);
}
/* 渐暗：99 → 0，每步 10ms */
for (int16_t d = 99; d >= 0; d--)
{
    pwm_set_duty((uint8_t)d);
    delay_ms(10);
}
```

总周期 = 200 步 × 10ms = **2s**。
